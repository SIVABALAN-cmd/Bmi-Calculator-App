 BMI Calculator App

A simple Body Mass Index (BMI) calculator built using React.

## 🚀 Features

- Enter height and weight to calculate BMI
- Displays BMI value and category (Underweight, Normal, Overweight, Obese)
- Responsive and clean UI

## 🛠️ Tech Stack

- React
- JavaScript
- CSS
